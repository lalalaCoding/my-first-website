HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications. Web browers receive HTML documents from a web server or from local storage and render then into multimedia web pages. HTML describes the structure of a web page semantically and originally includeed cues for the appearance of the document.
